***
## uname
```
uanme -a 打印所有信息
uanme -s 打印内核的名字
uanme -r 打印内核的发行
uanme -v 打印内核的版本
uname --help
```

***
## yum upgrade 和 yum update
```
yum update 升级所有包同时 也升级软件和系统内核
yum upgrade 只升级所有包，不升级软件和系统内核
```
